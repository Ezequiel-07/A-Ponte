# **App Name**: A Ponte

## Core Features:

- Busca e Filtragem de Empresas: Permite que os usuários busquem empresas com base na localização e nos códigos CNAE, possibilitando o filtro lógico para potencial de parceria.
- Recomendações Inteligentes de Conexão: Gera recomendações de parceria usando IA para analisar perfis de empresas com base em critérios objetivos, localização e perfil operacional, além de fornecer explicações legíveis sobre o raciocínio da IA.
- Integração de Dados CNPJ: Integra-se com a Brasil API para recuperar automaticamente os dados da empresa com base no CNPJ, facilitando a criação e atualização dos perfis da empresa.
- Autenticação Segura do Usuário: Implementa a autenticação segura do usuário usando o Firebase Auth, garantindo que apenas usuários autorizados possam acessar os recursos da plataforma.
- Ferramenta de Recomendação de Parceria: Um recurso alimentado por IA que fornece recomendações de parceria, usando o raciocínio para justificar cada sugestão com o contexto de negócios.
- Visualização de Recomendações: Exibe recomendações de parceria em um formato claro e compreensível, enfatizando os principais fatores e o raciocínio por trás de cada sugestão.
- Armazenamento de Dados: Os dados de conexões e perfis de empresas são salvos no banco de dados Firestore.

## Style Guidelines:

- Cor primária: Azul profundo (#1E3A8A) para evocar confiança e profissionalismo, adequado para uma plataforma B2B focada em conexões.
- Cor de fundo: Azul muito claro (#F0F9FF), uma versão dessaturada da primária, garantindo uma estética limpa e profissional.
- Cor de destaque: Roxo (#7E22CE), criando contraste e destacando elementos interativos importantes com um toque moderno.
- Emparelhamento de fontes: 'Space Grotesk' (sans-serif) para títulos e 'Inter' (sans-serif) para o corpo do texto para manter uma aparência moderna e profissional.
- Use ícones modernos e minimalistas para representar diferentes tipos de dados e ações, garantindo uma experiência de usuário limpa e intuitiva.
- Emprega um layout limpo e estruturado com hierarquia de informações clara, usando o espaço em branco de forma eficaz para evitar a desordem e garantir a legibilidade.
- Animações sutis com Framer Motion para fornecer feedback sobre as interações do usuário e guiar os usuários pela plataforma de forma intuitiva.
